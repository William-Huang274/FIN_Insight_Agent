# 历史脚本归档

这里放不再作为当前公开入口的历史脚本。归档脚本保留是为了查实现来源或复现实验，不代表还会随当前主链路持续维护。

## 版本目录

- `v0_1_free_query/`：早期自由问题回答质量打分脚本。当前自由问题解析入口已经收敛到 `scripts/eval_query_planner/`。
- `v0_1_vllm_diagnostics/`：早期 5090 / vLLM 环境诊断脚本。当前公开演示和 Workbench 不依赖这个诊断入口。

## 使用规则

- 新文档和用户入口不要引用这里的脚本。
- 如果归档脚本重新成为主线能力，先移回合适的稳定目录，再更新 `scripts/README.md` 和相关测试。
- 归档目录名必须带版本号，格式建议为 `v<major>_<minor>_<topic>/`。
